Class: Response

Response(windshaftSettings, serverResponse)

new Response(windshaftSettings, serverResponse)

Wrapper over a server response to a map instantiation giving some utility methods.
Parameters:
Name Type Description
windshaftSettings object Object containing the request options.
serverResponse string The json string representing a windshaft response to a map instantiation.

Methods

_getLayers()

Return the array with all the layers in the response

getBaseURL()

Build the base url to build windshaft map requests.

getLayerIndexesByType(Type)

Return the indexes of the layers for a certain type.
Parameters:
Name Type Description
Type string The type of the layers: mapnik, torque, plain, tiled.
Example
// layers = [ carto, carto, tiled, plain, tiled, torque];
getLayerIndexesByType('mapnik') // [0, 1]
getLayerIndexesByType('tiled') // [2, 4]
getLayerIndexesByType('torque') // [5]

getStaticBaseURL()

Build the base url for static maps.
